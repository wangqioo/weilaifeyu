# 「未来法域·全图」AI 辅助工作流 — Prompt 工具包

> 版本：v1.0 · 2026-03-23
> 适用：每周日晚 20:00 运行 Step 1，周一 08:00 运行 Step 2

---

## 核心原则

| 原则 | 说明 |
|---|---|
| 时效 > 热度 | 有热度的消息已经是二手了，「全图」的价值在于第一时间发现**还没热的信号** |
| 专业读者 | 律师、法务、监管从业者——他们不需要背景解释，需要信息密度 |
| 清单不评论 | 「全图」是认知地图，不是观点文章；每条一句话，事实为主 |
| 法律行业链接点优先 | 所有政治/经济/科技新闻，必须与**法律合规/诉讼/监管/立法实践**有直接关联才入选 |

## STEP 1 · 内容筛选 Prompt（周日 20:00 运行）

**使用方式：** 填入 `[本周日期]` 后，整段复制给 AI，让它完成网络搜索并输出结构化清单。

```
你是「未来法域」公众号「全图」栏目的研究助理。「全图」是一份面向中国法律职业群体的周一早读周报，
读者是律师、法务总监、监管从业者。

本周时间范围：[起始日期] 至 [截止日期]（北京时间）

你的任务是搜索并筛选上周全球重大事件，按以下四个板块各输出 4-6 条清单条目。

━━━━━━━━━━━━━━━━━━━━━━
板块定义与筛选标准
━━━━━━━━━━━━━━━━━━━━━━

【观势时局】全球政治、经济、军事、资本格局变动
  优先选：
  ✓ 战争/地缘冲突的阶段性节点（停火谈判、战线变化、制裁动作）
  ✓ 大国领导人峰会/会议/正式表态
  ✓ 宏观经济关键数据发布（IMF/WB/BIS/央行）
  ✓ 影响大宗商品/能源定价的结构性变化
  排除：
  ✗ 无实质进展的外交表态、日常贸易摩擦评论
  ✗ 单纯的股市涨跌数据（无结构原因）

【全球法讯】全球立法、司法判决、监管执法、合规要求
  优先选（这是「全图」核心价值所在，宁多勿少）：
  ✓ 新法律/法规正式生效、通过、公布（尤其 AI/数据/隐私/金融）
  ✓ 重大司法判决（尤其涉及平台责任、AI 著作权、跨境执法）
  ✓ 监管机构正式调查启动、处罚决定、指引发布
  ✓ 国际条约/协议签署（尤其投资仲裁、数据跨境、制裁体系）
  ✓ 立法机构草案一读/委员会通过（即使还未生效，预示方向）
  排除：
  ✗ 律所排名、法律行业商业新闻（无监管意义）
  ✗ 个案调解/非公开仲裁（无先例意义）

  重点关注地区：EU / US（联邦+各州）/ UK / 中国 / 东南亚 / 中东

【行业风向】能源、科技、AI、金融行业的结构性突破
  优先选：
  ✓ 重大技术发布（真实产品，非 PR 稿）——尤其对法律/合规产生实际影响的
  ✓ 行业重组：并购、破产、市场退出
  ✓ 基础设施投资（数据中心、能源协议、供应链）
  排除：
  ✗ AI 产品功能更新、app 版本发布（无结构意义）
  ✗ 融资轮次（除非体量改变行业格局）
  ✗ 企业 ESG 声明、公关稿件

【暗涌】低热度但高时效的法律行业链接点
  这是最难搜索但最有价值的板块。具体含义：
  ✓ 某立法/判决刚生效，业界尚未广泛讨论，但未来 6-12 个月会产生合规压力
  ✓ 监管机构的"软信号"——指引草案、问询函、研讨会纪要
  ✓ 跨境合规新盲区：某国制裁豁免创造的法律先例、多边条约的执行漏洞
  ✓ 能源/科技新合同模式引发的不可抗力/仲裁新问题
  搜索策略（必须主动搜，而非等热点浮现）：
  - EU Official Journal (eur-lex.europa.eu) 本周新发布法规
  - US Federal Register 本周新规
  - OFAC/BIS/FinCEN 本周执法动作
  - 各国立法机构本周委员会投票记录
  - ICSID/ICC 本周新注册仲裁（如有公开信息）

━━━━━━━━━━━━━━━━━━━━━━
输出格式（严格遵守）
━━━━━━━━━━━━━━━━━━━━━━

每条格式：
**[关键词/标题]** — 一句话事实陈述（不超过40字）。地区/机构标签（如：EU/US/中国）。

示例（正确）：
**EU AI Act 执法倒计时** — 欧洲议会研究室3月18日确认，GPAI模型义务将于8月2日正式适用，企业合规窗口仅剩135天。EU。

示例（错误，排除）：
**ChatGPT 发布新功能** — OpenAI 本周更新了 GPT-4o 的语音功能，用户反馈良好。（无监管意义，排除）

━━━━━━━━━━━━━━━━━━━━━━
额外要求
━━━━━━━━━━━━━━━━━━━━━━

1. 每条附信源简称（如 Reuters / FT / EUR-LEX / Federal Register / 财新）
2. 如果某板块真实符合标准的新闻不足4条，宁可留空或标注"本周该板块无重大动态"，不要用低质量新闻凑数
3. 最后输出一行「本周主线」：用一句话（不超过60字）概括本周最值得关注的跨板块主题

请开始搜索并输出结果。
```

## STEP 2 · 格式化 Prompt（周一 08:00 运行）

**使用方式：** 将 Step 1 的输出粘贴到 `[Step 1 输出内容]` 处，AI 会直接生成微信 HTML。

```

本期信息：
- 栏目：全图 PANORAMA
- 期号：Vol.[XXX]
- 时间范围：[起始日期] — [截止日期]
- 本周主线：[从 Step 1 结果复制]

Step 1 筛选结果：
[粘贴 Step 1 输出]

━━━━━━━━━━━━━━━━━━━━━━
HTML 格式要求（严格遵守）
━━━━━━━━━━━━━━━━━━━━━━

1. 所有样式必须内联（无 <style> 标签），微信编辑器只支持内联 style
2. 主色：#2d5a27（松绿）；背景：#f8f7f4；正文：#333333；次要：#888888
```

## STEP 3 · 关键词坐标图更新（周一 08:00，2分钟完成）

1. 打开 `outputs/chart-generator.html`
2. 编辑顶部 `ISSUE` 对象（更新 `vol` 和 `period`）
3. 编辑 `KEYWORDS` 数组（参考 Step 1 筛选结果，格式：`["标签", x坐标, y坐标]`）
   - X轴：-2（中国内）→ 0（跨境）→ +2（国际）
   - Y轴：-2（政治经济）→ 0（法律监管）→ +2（科技产业）
4. 保存 → 浏览器刷新 → 点「下载 PNG」
5. 上传至微信素材库 → 插入文章 → 替换占位卡片

## 常见问题 & 处理规则

**Q: AI 给的某条信息无法核实来源怎么办？**
A: 直接删除。「全图」不发布无法核实的信息，宁缺勿滥。

**Q: 某周某板块确实没有重大新闻怎么办？**
A: 保留板块标题，写「本周暂无重大动态」。不要用低质量新闻凑数。

**Q: 某条信息跨越多个板块怎么归类？**
A: 按**最直接的法律行业影响**归类。例如「NVIDIA 反垄断调查启动」放「全球法讯」而非「行业风向」。

**Q: 「暗涌」板块搜不到怎么办？**
A: 必须主动去官方来源检索，而非依赖新闻聚合。优先级：
   1. EUR-LEX 本周 OJ（官方公报）
   2. US Federal Register 本周
   3. OFAC.treasury.gov 本周执法行动
   4. 中国全国人大官网/国务院政策发布
